Welcome to Java Class Parts Home Page.

The Java Class Parts Game is a simple Game to help novice programmer learn the basic parts of a Java Class:

Here is an active version:Java Parts Game

About the Game

There are three web pages that a user can select:
  • An example page which just shows the user what the name of the parts of a class are
  • A tutorial page that allows a user to click on a class parts and it tells them what the part is called
  • A game page the requests the player click on the indicated part
